# Instructions d'installation

Pour terminer la configuration de la base de données, exécutez :

```bash
npm install @duckdb/duckdb-wasm apache-arrow@14
```

Ou avec yarn :

```bash
yarn add @duckdb/duckdb-wasm apache-arrow@14
```

Ces dépendances sont nécessaires pour le fonctionnement de DuckDB WASM.
